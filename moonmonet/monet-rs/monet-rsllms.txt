# MonetRs (Rust)
A Rust library implementation of Android 12's Monet (Material You) color generation system. Generates dynamic color palettes based on a seed color.

## Dependencies
**Cargo.toml**:
```toml
[dependencies]
monet_rs = "0.1" # Check for latest version
```

**Imports**:
```rust
use monet_rs::theme::dynamic_color_scheme::DynamicColorScheme;
use monet_rs::theme::target_colors::TargetColors;
use monet_rs::colors::srgb::Srgb;
```

## Boilerplate
```rust
use monet_rs::theme::dynamic_color_scheme::DynamicColorScheme;
use monet_rs::theme::target_colors::TargetColors;
use monet_rs::colors::srgb::Srgb;

fn main() {
    // Configuration
    let chroma_multiplier: f64 = 1.0; // Range: 0.5 - 2.0 (typically)
    let is_dark_mode: bool = true;
    
    // Seed Color (RGB)
    let r = 0.0; 
    let g = 0.73; 
    let b = 0.75;
    let seed_color = Srgb::new(r, g, b);

    // Generate Scheme
    let scheme = DynamicColorScheme::new(
        TargetColors::new(chroma_multiplier),
        seed_color,
        chroma_multiplier,
        is_dark_mode
    );

    // Access specific color (Returns u32 ARGB)
    let primary_argb: u32 = scheme.get_accent1(500).quantize8();
    
    println!("Primary Color ARGB: {:08X}", primary_argb);
}
```

## API Reference

### Initialization
```rust
DynamicColorScheme::new(
    target_colors: TargetColors, 
    seed_color: Srgb, 
    chroma: f64, 
    is_dark: bool
)
```

### Palettes (Methods)
All methods take a `shade` (int) argument.
*   `scheme.get_accent1(shade)`: Primary accent.
*   `scheme.get_accent2(shade)`: Secondary accent.
*   `scheme.get_accent3(shade)`: Tertiary accent.
*   `scheme.get_neutral1(shade)`: Backgrounds/Surfaces.
*   `scheme.get_neutral2(shade)`: Outlines/Borders.

### Supported Shades (Integers)
Passed as arguments to the palette methods.
*   `0` (Lightest)
*   `10`
*   `50`
*   `100`
*   `200`
*   `300`
*   `400`
*   `500` (Mid-tone)
*   `600`
*   `700`
*   `800`
*   `900`
*   `1000` (Darkest)

### Output Format
The palette methods return a color object that must be converted.
*   `.quantize8()`: Returns `u32` in **ARGB** format.

## Syntax & Patterns
```rust
// Pattern: Initialize -> Get Color Object -> Quantize to u32
let color_u32 = scheme.get_neutral1(900).quantize8();
